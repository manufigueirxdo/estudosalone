# 🚦 Módulo 02: Condicionais e Regras de Negócio

O objetivo deste módulo foi implementar lógica de decisão (`if/else/elif`) e operadores de comparação (`>`, `<`, `==`, `!=`) para proteger sistemas contra entradas inválidas e automatizar regras corporativas.

## 🎯 Desafio Resolvido: Sistema de Folha de Pagamento
Desenvolvimento de uma calculadora de salário líquido com aplicação automática de bônus baseada em faixas salariais.

## 🛡️ Foco em Segurança e Validação
* **Prevenção de Bugs Lógicos:** Implementação de uma barreira condicional que impede o cálculo de valores caso o usuário digite um salário negativo ou zerado, protegendo o caixa da empresa contra vazamentos financeiros por erro de digitação.
* **Arquitetura Limpa:** Uso de condicionais aninhadas para garantir que variáveis de bônus só consumam memória do servidor se o funcionário realmente tiver direito ao benefício.
