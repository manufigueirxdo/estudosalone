# 🚀 Módulo 05: Automação e Processamento em Lote

## 🎯 Objetivo do Módulo
Automatizar cálculos financeiros e processamentos repetitivos, eliminando o trabalho manual e o erro humano através de iteração de dados.

---

## 💻 Projetos Desenvolvidos

### 1. Motor de Precificação Dinâmica (`simulador_parcelas.py`)
- **O problema que resolve:** Elimina a demora no atendimento ao cliente, calculando e exibindo instantaneamente todas as opções de parcelamento de uma venda com juros/frete embutidos.
- **Ferramentas utilizadas:** Laço `for` e função `range()`.

### 2. Simulador de Cenários de Câmbio (`simulador_dolar.py`)
- **O problema que resolve:** Traz previsibilidade financeira para o freelancer, mapeando automaticamente quanto um projeto internacional vai render em Reais dependendo da variação do dólar.
- **Ferramentas utilizadas:** Multiplicação de variáveis iteráveis e formatação de saída.

### 3. Processador de Inadimplência (`processador_dividas.py`)
- **O problema que resolve:** Substitui a cobrança manual de boletos atrasados. O sistema varre uma lista de devedores e aplica a multa exata para cada um em milissegundos.
- **Ferramentas utilizadas:** Manipulação de Listas `[ ]` e formatação financeira `:.2f`.

---

## 🛠️ Stack Tecnológica Dominada
- Iteração de dados com Laço `for`
- Geração de sequências numéricas (`range`)
- Estruturas de Dados em lote (Listas)
- Formatação de Ponto Flutuante para padrão monetário
