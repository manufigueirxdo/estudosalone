# Módulo 01: Fundamentos e Matemática Financeira 🧮

Este módulo contém a base do meu treinamento em Engenharia de Software com Python, focado em traduzir regras de negócios reais para lógica de programação.

## 🚀 Microserviços Desenvolvidos

### 1. Calculadora de Câmbio no Checkout (`calculadora_cambio_checkout.py`)
Simula o backend de um e-commerce internacional.
* **Problema resolvido:** Converte compras em dólar para real aplicando uma taxa de câmbio estática e adicionando automaticamente 6% de IOF.
* **Habilidades aplicadas:** Tipagem de dados (`float`), operadores matemáticos e formatação de strings de saída (`f-strings`).

### 2. Motor de Salário Proporcional (`calculadora_salario_proporcional.py`)
Simula o backend de um sistema de Recursos Humanos (RH).
* **Problema resolvido:** Calcula o pagamento fracionado de um funcionário com base nos dias efetivamente trabalhados no mês e extrai a alíquota de 8% para depósito do FGTS.
* **Habilidades aplicadas:** Ordem de precedência matemática, conversão mista de dados (`int` e `float`) e arquitetura de cálculo sequencial.
