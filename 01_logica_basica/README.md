# 🧠 Módulo 01: Lógica Básica e Tipagem de Dados

Neste primeiro módulo, o foco foi dominar a sintaxe base do Python e a manipulação de diferentes tipos de dados (`int`, `float`, `string`) aplicados a problemas reais de negócios.

## 🎯 Desafios Resolvidos
1. **Sistema de ROI:** Script para calcular faturamento, lucro líquido e retorno sobre investimento em campanhas de anúncios.
2. **Automação Logística:** Algoritmo de otimização de frotas utilizando operadores de divisão inteira (`//`) e resto (`%`).
3. **Conversor de Moedas:** Manipulação de entradas do usuário para cálculos de conversão cambial.

## 💡 Aprendizados Técnicos
* Captura e conversão de dados do usuário (Casting).
* Formatação de saída limpa com `f-strings`.
* Resolução de erros de sintaxe e tipagem em tempo de execução.
