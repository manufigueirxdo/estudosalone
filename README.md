## 📂 Projetos e Scripts Desenvolvidos

Nesta primeira fase, foquei em **Lógica de Programação**, **Casting de Tipos** e **Operadores Aritméticos**:

1. **Calculadora de ROI (Return on Investment):** Script que calcula faturamento, lucro líquido e retorno percentual.
2. **Automação de Logística:** Algoritmo que utiliza divisão inteira (`//`) e resto (`%`) para otimização de carregamento de frotas.
3. **Conversor de Moedas e Tributação:** Manipulação de `floats` para cálculos financeiros e conversão de câmbio.
4. **Formação de Preço com Margem de Lucro:** Lógica para cálculo de preço de venda com base em custo e metas de lucro.

## 🧠 Lições de Engenharia Aprendidas

> "O código é apenas uma ferramenta; a engenharia está em como você resolve o problema."

* **Debugging:** Aprendi a interpretar erros de `SyntaxError`, `NameError` e `ValueError` no terminal.
* **Clean Code:** Uso de `f-strings` para melhor legibilidade e organização de código em `snake_case`.
