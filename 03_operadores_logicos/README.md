# Módulo 03: Operadores Lógicos 🧠

Este módulo consolida o uso de múltiplas condições simultâneas para validação rigorosa de regras de negócios.

## 🚀 Microserviços Desenvolvidos

### 1. Motor de Aprovação de Empréstimos (`motor_aprovacao_emprestimo.py`)
* **Problema resolvido:** Sistema de análise de crédito bancário que avalia simultaneamente a renda mínima exigida e o teto de comprometimento salarial (30%).
* **Habilidades aplicadas:** Operador lógico `and` para cenários de alta restrição (onde todas as condições devem ser estritamente verdadeiras para a execução do bloco de aprovação), e pré-cálculo de variáveis dinâmicas.
